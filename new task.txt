This project is about building a multi-agent RAG (Retrieval-Augmented Generation) chatbot that can read two PDFs, understand them, and answer user questions with very high accuracy.

Unlike normal PDF-QA chatbots that use a single agent, this system uses five coordinated AI agents, all connected using LangGraph, to ensure structured reasoning and minimal hallucination.
